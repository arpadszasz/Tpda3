Introducere

TPDA (Tiny Perl Database Application) este infrastructura pe care se pot clădii rapid și eficient aplicații de baze de date.

O aplicație este formata dintr-o bază de date cu una sau mai multe tabele și câte un Ecran pentru fiecare tabel principal.

Ecranul este un element grafic de tip fereastră care conține alte elemente grafice cum ar fi câmpuri pentru editarea textelor, butoane de diverse feluri, tabele, ș.a.m.d.

Aplicația este formată din două entități distincte, prima este sistemul de gestiune al bazei de date (SGBD) care formează partea de server și a doua, adică partea de client, aplicația TPDA.
